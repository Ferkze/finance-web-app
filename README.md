## Como rodar esse negócio

### Requisitos
- Git +2.x
- Node.js +8.x
- NPM +5.x
- Yarn +1.2
  
### Passo a passo de instalação
1. Instalação de dependências
```
yarn install
```
2. Compila e roda a aplicação para desenvolvimento
```
yarn run serve
```
3. Compila e minifica para produção
```
yarn run build
```
